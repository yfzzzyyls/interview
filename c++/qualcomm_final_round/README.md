# Qualcomm Final Round C++ Practice

Use this directory for focused C++ practice for the Qualcomm CPU Performance Modeling final round.

## Folders

- `01_basics_coding/`: small C++ syntax and fundamentals blocks.
- `02_fluency_bridge/`: short bridge drills for C++ mechanics used in simulator code.
- `03_microarchitecture_coding/`: simulator and CPU microarchitecture coding drills.

## Compile Rule

Use:

```bash
g++ -std=c++17 -Wall -Wextra -pedantic file.cpp -o file
```

For interview practice, every file should have a small `main()` or clear test harness unless it is intentionally written in LeetCode class format.
